
# Xylophone App 🎵

A simple and interactive Xylophone application built using **Swift** and **UIKit** in **Xcode**. This app allows users to play different musical notes by tapping colorful keys, providing a fun and engaging musical experience.

## ✨ Features

* Play different musical notes by tapping the xylophone keys
* Colorful and responsive user interface
* Smooth audio playback
* Built entirely with UIKit
* Beginner-friendly project structure

## 🛠️ Built With

* **Language:** Swift
* **Framework:** UIKit
* **IDE:** Xcode
* **Architecture:** MVC


## 🎯 Learning Objectives

This project was created to practice and strengthen knowledge in:

* Swift programming
* UIKit components
* Audio playback in iOS
* Auto Layout
* Storyboard-based UI development
* MVC design pattern
* Handling user interactions
